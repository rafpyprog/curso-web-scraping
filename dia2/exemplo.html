Introdução



BeautifulSoup é um pacote para extração de dados de HTML e XML. A documentação pode ser encontrada neste link.
  • Criado em 2004
  • Extração de dados em arquivos XML e HTML
  • Funciona com diferentes parsers
  • Linguagem clara e intuitiva
  • Detecação e conversão automática da codificação das strings
  • Ótima capacidade para lidar com código HTML mal formatado
